Taps (Data Sources)
-------------------

PipelineWise can replicate data from the following data sources:

* MySQL (:ref:`tap-mysql`)
* MariaDB (:ref:`tap-mysql`)
* PostgreSQL (:ref:`tap-postgres`)
* Zendesk (:ref:`tap-zendesk`)
* Kafka stream (:ref:`tap-kafka`)
* CSV file on S3 (:ref:`target-s3-csv`)
